# BMI Web App

A simple and responsive BMI (Body Mass Index) calculator web application built using React.

## Tech Stack
- React.js
- Vite
- JavaScript
- CSS
- Node.js (for development environment)

## Features
- Calculate BMI based on height and weight
- Displays BMI category (Underweight, Normal, Overweight, Obese)
- Clean and responsive UI
- Fast development setup with Vite

## Getting Started

### Prerequisites
- Node.js installed

### Installation

npm install
